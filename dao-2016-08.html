21. An amount invested 22,200 at 8. 4% from March 08, 2016 to August 22, 2016. For numbers 1-2. 1. Find the ordinary interest using actual time *(Round off to at least 2 decimal places)2 points2. Find the future value of ordinary interest using approximate time *(Round off to at least 2 decimal places).


Answer:

To find the ordinary interest, we first need to calculate the daily interest rate. The annual interest rate of 8.4% can be converted to a daily interest rate by dividing it by 365 (the number of days in a year).

Daily interest rate = 8.4% / 365 = 0.02297

Next, we need to find the number of days between March 08, 2016 and August 22, 2016.

Number of days = (August 22, 2016 - March 08, 2016) = (22 - 8) = 14

Finally, we can find the ordinary interest by multiplying the daily interest rate by the number of days by the amount invested, which is 22,200.

Ordinary interest = 0.02297 * 22,200 * 14 = 711.2558

Rounding off to 2 decimal places, the ordinary interest is 711.26.

To find the future value of ordinary interest, we first need to add the ordinary interest to the original amount invested.

Future value of ordinary interest = 22,200 + 711.26 = 22,911.26

Rounding off to 2 decimal places, the future value of ordinary interest is 22,911.26.
