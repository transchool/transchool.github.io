13. who can help me with my math assignment the topic is about picewise function the problem is " A videoke machine can be rented for P1000 for three days, but for the fourth day onwards, an additional cost of P100 per day is added . Represent the cost of renting a videoke machine as a piecewise function of the number of days it is rented.​


Answer:

a) f(x) = 1,000;  0 < x ≤ 3

b) f(x) = 1,000 + 100(x-3);   x > 3

Step-by-step explanation:

Flat or fixed rate for 3 days: Php1,000

Additional rate per day after 3 days: Php100/day

Number of days: x

Total Charges: f(x)

Equation:

Piecewise function:

a) f(x) = 1,000;  0 < x ≤ 3

b) f(x) = 1,000 + 100(x-3);   x > 3

Application:

a)

Number of days rented: 2 days

The function to be used is (a) because x = 2 days

f(2) = 1,000

Total charges is Php 1,000.

b)

Number of days = 5

The function to be used is (b).

f(5) = 1,000 + 100 (5-3)

f(5) = 1,000 + 100(2)

f(5) = 1,000 + 200

f(5) = 1,200

Total charges is Php1,200.
