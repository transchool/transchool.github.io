Paliwanag:

Pananalita ay isang aktibidad na madalas ginagawa. Hindi lamang sa mga opisyal ng estado. Maaaring gawin ang mga talumpati kahit saan anumang oras. Sa isang talumpati, maaaring ipabatid ng isang tao ang iba't ibang bagay na may maraming layunin.

1. Impormatibong Talumpati

Isang uri ng talumpati na alam ng maraming tao ang talumpating nagbibigay-kaalaman. Ang talumpating nagbibigay-kaalaman ay isang talumpati na may layuning maghatid ng mahahalagang kaalaman o impormasyon. Ang impormasyong ito ay ibibigay sa tagapagbalita o tagapakinig. Ito ay nilayon upang malaman ng nakikinig ang ipinahihiwatig. Dagdag pa rito, upang maunawaan ng nakikinig ang mga impormasyong ihahatid. Inaasahan din na matatanggap ng mga tagapakinig ang mga impormasyong ipinarating.

2. Talumpating argumentative

Ang susunod na uri ng talumpati ay talumpating argumentative. Ang talumpati sa argumento ay isang talumpating naglalaman ng ilang bagay. Gaya ng mga proposisyon, argumento, datos o dahilan. Ang mga bagay na ito ay nagsisilbing pagtanggi o pagsuporta sa isang pahayag.

3. Recreative Speech

Ang susunod na uri ng talumpati ay recreational speech. Ang mga malikhaing talumpati ay madalas ding tinutukoy bilang mga talumpati ng pamilya. Ang ganitong uri ng pananalita ay karaniwang magpapakita ng kagalakan.

4. Mapanghikayat na Talumpati

Ang ikaapat na uri ng talumpati ay isang mapanghikayat na pananalita. Ang layunin ng isang mapanghikayat na talumpati ay naimpluwensyahan ng nakikinig. Ang persuasion ay isang proseso upang maimpluwensyahan ang opinyon.
