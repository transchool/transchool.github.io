19. which of the following situation illustrates combination?A.arranging books in shelfB.drawing names from a box containing 200 namesC.forming different numbers from 5 given digitsD.forming plate numbers of vehicles​


COMBINATION

B. Drawing  names from a box containing 200 names.

A combination is a mathematical technique for calculating the number of possible arrangements in a set of objects where the order of the elements doesn't matter. You can put the things in whatever order you like in a combination. Permutations and combinations are often mixed up.

FORMULA FOR COMBINATION

The following is the mathematical formula for computing the number of different configurations by selecting only a few pieces from a collection with no repetition:

             [tex]nCr=\frac{n!}{r!(n-r)!}[/tex]

nCr= the number of combination

n=total number of items in the collection

r=the number of objects from the collection that can be chosen

EXAMPLE OF COMBINATION

 A combination is a selection of all or part of a collection of things, regardless of their order of selection. Let's say we have three letters: G, H, and I. We can wonder how many different ways we can choose two letters from that list. Each option is an example of a combination.

THE DIFFERENCE OF COMBINATION AND PERMUTATION

    Many people are confused about permutation and combination. So this is the key difference to identify if it is a combination or permutation.

Permutation refers to the various methods of placing a group of objects in a sequential sequence. While Combination is one of numerous methods for selecting items from a big group of objects without regard for their order. https://brainly.ph/question/2475884From a single combination, several variations can be created.
