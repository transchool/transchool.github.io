1. ano ang email adress


Answer:

Kinikilala ng isang email address ang isang kahon ng email kung aling mga mensahe ang naihatid. Habang ang mga system ng maagang pagmemensahe ay gumamit ng iba't ibang mga format para sa pagtugon, ngayon, ang mga email address ay sumusunod sa isang hanay ng mga tukoy na panuntunang orihinal na na-standardize ng Internet Engineering Task Force noong 1980, at na-update ng RFC 5322 at RFC 6854.

8. ANO ANO ANG TATLONG BAHAGI NG EMAIL ADDRESS? Brainly.com


Ang email address ay ang pagkakakilanlan ng isang email. Tumutukoy ito sa pangalan ng email kung saan dapat isend o ipadala ang isang email message. Para itong address ng tirahan. Kumbaga, isa itong destinasyon at pinagmulan ng isang email message. Mayron itong 3 bahagi. Ito ay ang mga sumusunod:

name@emailprovider.com

1. Username - ang username ay ang unique o kakaibang name na siyang akda o gawa ng user na gagamit ng email. Iisa lamang ang username at wala itong katulad sa isang email.  

2. @ - ang @ sign ay nagsisilbing divider o naghihiwalay sa username at domain name.

3. Domain / Email service provider - Ang domain name ang siyang tutukoy sa email service provider. Gaya ng nasa halimbawa, ang domain o email provider natin ay ang emailprovider.com. Kadalasan, ito ay isang kumpanya na syang naghahawak ng ating mga emails na sinesend at narerecieve online.
