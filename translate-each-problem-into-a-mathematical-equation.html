1. Translate each problem into a mathematical equation. ​


1.) Let's say, x = the price of the old model

GIVEN:

Price of new 32"LED tv = ₱29,078.00

Price of old model tv = x

SOLUTION:

2x - 15,500 = 29,078

2x = 29,078 + 15,500

2x = 44,578

2x/2 = 44,578/2

x = ₱22,289

Therefore the price of the old model is ₱22, 289

2.) FORMULA OF THE PERIMETER OF RECTANGLE:

P = 2(l + w)

GIVEN:

P = 96

l = 2w

w = w

SOLUTION:

FIRST, FIND THE VALUE OF THE WIDTH:

P = 2 (l + w)

96 = 2 (2w + w)

96 = 2 (3w)

96 = 6w

96/6 = 6w/6

w = 16

the width is 16

SECOND, SUBSTITUTE THE VALUE OF WIDTH TO IDENTIFY THE VALUE OF LENGTH:

l = 2w

l = 2(16)

l = 32

the length is 32

Therefore, the dimensions of the rectangle are 16 × 32.

CHECK:

P = 2 (l + w)

96 = 2 ( 32 + 16 )

96 = 2 (48)

96 = 96, check✓ the solution was correct.
