5. anu ang ibig sabihin ng ibig sabihin


Ibig-sabihin

Ang kahulugan ng salitang ibig-sabihin, ay ang salita o mga salita na kasing-kahulugan ng isang bagay, tao, pangyayari o kaganapan. Ito ang nagsasaad kung ano nga ba ang isang bagay.

Halimbawa ng pangungusap

Nais malaman ni Grace, ang ibig-sabihin ng mga mahihirap na salita, kaya humiram siya ng diksyunaryo sa silid-aklatan ng paaralan.

30. Ano ibig sabihin ng FLOWCHART?Ano ang ibig sabihin ng FISHBONE DIAGRAM?Ano ang ibig sabihin ng CYCLE DIAGRAM?Ano ang ibig sabihin ng WORD PROCESSING TOOL?​


Answer:

Word processing tool

-ang ibig sabihin ng Word Processing Tool ay nakakatulong sa isang entrepreneur sa paggawa ng mga ulat at mga liham

-ang isang ulat tungkol sa kalagayan ng mga benta, puhunan, kita o lugi ay nagiging mas makahulugan pa kung lalagyan ng diagram.

Flowchart

Ay isang uri ng diagram na nagpapakita ng sunod sunod na hakbangin na nagsasalarawan ng daloy ng trabaho o proseso.

Ito ay kalimitang ginagamit sa pagsusuri, pag didisenyo maging pamamahala ng mga proseso o programa sa anumang larang.

Venn Diagram

Ito ay nagpapakita ng mga posibleng lohikal na pagkakaiba at pagkakatulad ng mga bagay. Si John Venn ang unang nagpakilala ng Diagram noong 1880

Fish Bone

Ito ay kilala rin bilang Ishikawa Diagram.

Ginagamit ito upang ipakita ang sanhi at epekto ng isang pangyayari.

Malaki ang tulong nito sa pagsusuri sa mga naging ugat o dahilan ng problema.

Cycle Diagram

Ito ay nagpapakita ng pagkakaugnay ng mga bagay sa isang paulit ulit na proseso.

Ito ay kalimitang gumagamit ng mga circular arrows upang iugnay ang isang yugto sa susunod na yugto.
