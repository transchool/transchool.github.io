18. Activity 2. Simplify Me.Direction: Simplify the following expressions:​


Activity 2. Simplify Me.

1. [tex]\frac{2x}{6x}[/tex]

Common factor between the numerator and denominator is [tex]2x[/tex]. Divide both by [tex]2x[/tex].

[Sol]

[tex]\frac{2x}{6x}\\=\frac{1}{3}[/tex]     Answer: 1/3

2. [tex]\frac{6y+9z}{12}[/tex]

Common factor is [tex]3[/tex]. Divide all terms by [tex]3[/tex].

[Sol]

[tex]\frac{6y+9z}{12} \\=\frac{2y+3z}{4}[/tex]     Answer: [tex]\frac{2y+3z}{4}[/tex]

3. [tex]\frac{10b-2bc}{5}[/tex]

There is no common factor in this expression. It is already in simplest form unless you want to divide (i.e. set in or break in) it into two fractional parts: [tex]\frac{10b}{5}\ and\ \frac{-2bc}{5}[/tex]. Simplify [tex]\frac{10b}{5}[/tex] by dividing the numerator and denominator each by 5. The simplified form will be [tex]2b-\frac{-2bc}{5}[/tex].

4. [tex]\frac{36d^{8}}{6d^{4}} \\[/tex]

Common factor is [tex]6d^{4}[/tex].

[Sol]

[tex]\frac{36d^{8}}{6d^{4}}\\=6d^4[/tex]     Answer: [tex]6d^{4}[/tex]

5. [tex]\frac{5}{25e^{3}}[/tex]

Common factor is [tex]5[/tex].

[Sol]

[tex]\frac{5}{25e^{3}}\\=\frac{1}{5e^{3}}[/tex]     Answer: [tex]\frac{1}{5e^{3}}[/tex]

In simplifying rational algebraic expressions, divide them by the common factor between the numerator and denominator. The factor must be applicable for all terms. Example:

[tex]\frac{2a^{2}+8}{16}[/tex]     The common factor is 2. Divide all terms by 2.

[Sol]

[tex]\frac{2a^{2}+8}{16}\div\frac{2}{2}\\=\frac{a^2+4}{8}[/tex]     The simplified form is [tex]\frac{a^2+4}{8}[/tex].
