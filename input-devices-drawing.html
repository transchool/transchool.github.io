18. Differentiate INPUT DEVICES and OUTPUT DEVICES​


An input device sends information to a computer system for processing, and an output device reproduces or displays the results of that processing. Input devices only allow for input of data to a computer and output devices only receive the output of data from another device.

19. Describe input devices output devices and storage device​


Input, used to interact with, or send data to the computer (mouse, keyboards, etc.) Output, which provides output to the user from the computer (monitors, printers, etc.) Storage, which stores data processed by the computer (hard drives, flash drives, etc.)


20. 1. is a computer input device that enables a user to hand-draw images, animations and graphics, similar to the way a person draws images with a pencil and paper.​


A graphic tablet is a computer input device that enables a user to hand-draw images, animations, and graphics, similar to the way a person draws images with a pencil and paper.

It typically consists of a flat surface (tablet) on which the user can draw or make gestures, and a stylus or pen-like device that is used to interact with the tablet. The stylus or pen can be used to draw, paint, or select items on the screen, and the tablet can detect the stylus or pen's position, pressure and tilt.

Graphic tablets are commonly used by digital artists and designers, as they allow for more natural and precise input than a mouse or touchpad. They can also be used for handwriting recognition, digital signatures, and other tasks that require accurate input. Some graphic tablets also feature touch-sensitive controls for additional functionality.
