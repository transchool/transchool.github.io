7. 1. A man deposited in his savings account a fixed sum severy month. At the end of n months he has saved P 3000,then n variesas S.


Answer:

To solve this problem, you can use the following formula:

P = s * n + 3000

Where P is the total amount saved after n months, s is the fixed sum deposited every month, and n is the number of months.

To find the value of n, you can rearrange the formula to solve for n:

n = (P - 3000) / s

Substitute the given values for P and s into the formula to find the value of n:

n = (3000 - 3000) / s

= 0 / s

= 0

Therefore, the value of n varies as S.
