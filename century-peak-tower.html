4. From a point, 80 m from the base of a tower, the angle of elevation to the top of the tower is 28°. How tall is the tower?


Let the height of the tower be x.

We can solve this problem using one of the trigonometric ratios (SohCahToa). Since the missing value is opposite to the angle, and the adjacent length is given, we can use tangent.

tan θ = opposite/adjacent

tan 28˚ = x/80m

(tan 28˚)(80m) = x

x = 42.53675453...m

x ≈ 42.54 meters

Based on the problems, the solutions are:

- Formulated:

= [tex](45-20):45[/tex]

= [tex]\frac{25}{45}[/tex]

= [tex]\frac{5}{9}[/tex]

- Converted to percentage:

[tex]\frac{5}{9}[/tex] = 0.5555

0.5555 x 100 % = 55.55 %

About Percentage

The percentage represents the difference in value between two numbers when that difference is stated as a percentage of each number's own average. For the purpose of calculating the percentage of a number that is contained within another number, the following formula may be utilized:

          Percentage = (Original number/Another number) x 100 %

In mathematics, the term "percentage" can also refer to a number or ratio that reflects a fraction of the value "one hundred." Percent denotes per 100. It does not come with any of those units that you specified. The following steps will walk you through determining the appropriate percentage:
