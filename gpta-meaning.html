3. The GPTA officers can paint a classroom in 3 hours, while Class officers can paint the same classroom in 5 hours. If the GPTA and Class officers work together, how long will i take them to paint the classroom?​


Answer:

1.875 hours

Step-by-step explanation:

Use the Working Together Formula:

1/T = 1/A + 1/B

where A = time of GPTA finishing the work

B = time of Class Officers finishing the work

T = total time of them working together

1/T = 1/3 + 1/5

1/T = (5+3)/15

1/T = 8/15

Cross multiply:

(8)(T) = (1)(15)

8T = 15

T = 15/8 or 1.875 hours

19. The School GPTA officers want to give t-shirts to all the students for the foundation day. They found a supplier that sells t-shirts for ₱180 per piece. The supplier agreed to give the t-shirt for ₱145 for a bulk order of 100 shirts or more. Use a piecewise function to express the cost in terms of the number of t-shirt purchases.Could you please explain in easiest way.​


Answer:

Let x be the number of t-shirts purchased. Then we can define the cost of x t-shirts as a piecewise function:

C(x) = {

₱180x, if x < 100

₱145x, if x ≥ 100

}

This function says that if x is less than 100, then the cost of x t-shirts is ₱180x. But if x is greater than or equal to 100, then the cost of x t-shirts is ₱145x.

This is because if the number of t-shirts purchased is less than 100, the supplier will charge the regular price of ₱180 per t-shirt. But if the number of t-shirts purchased is 100 or more, the supplier will offer a bulk discount of ₱35 per t-shirt, resulting in a price of ₱145 per t-shirt.

Therefore, the piecewise function C(x) represents the cost of purchasing x t-shirts from the supplier, taking into account the bulk discount for larger orders.
