7. Write a function for this supermarket promo "Buy 5,get each one after that at half price"


We want to find a function y = f(x), where y is the total price, and x is the number of items bought. Let's also say that we have a constant p, where p is the price per item.

If x <= 5, then y = px, right? Since the price of everything is just the product of the number of items and the price per item. Even if x=5, we won't benefit from the promo yet.

If x>5, we have y= 5p + 1/2p(x-5), where 5p is the price of the first 5 items (at full price), and 1/2p(x-5) is the rest of the items (at half price). Note that we used (x-5) instead of x, so that we don't count the first 5 items, since they are already included in the "5p" part.

Then, that piecewise function is already the answer:

f(x) = { px, x<=5

{ 5p + 1/2p(x-5), x>5
