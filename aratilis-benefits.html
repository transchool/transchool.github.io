Given S = {apple, aratilis, atis, avocado,} determine the n(S)

a. {aratilis, atis, apple, avocado, }

b. 4 <=== ANSWER

c. aratalis, apricot, atis, & avocado

d. Ø

Explanation:

- Cardinality is the number of elements a set has.

- It is denoted by the symbol n(A) or |A|, where A is the name of the set.

S = {apple, aratilis, atis, avocado,}

1st element = apple

2nd element = aratilis

3rd element = atis

4th element = avocado

1 + 1 + 1 + 1 = 4

n(S) = 4
