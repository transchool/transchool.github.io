If CDE = EDF then EFH = ?

Now, let's turn these key clues into mathematical numbers, say the numbers 1, 2, and 3. Then it will be like this:

C D E = E D F

1 2 3 = 3 2 4

As seen in the table above, we suppose the CDE as the numbers 1, 2, and 3, with the following description :

C is represented the number 1 D represented the number 2 E represents the number 3

From there, the value of EDF can be found, namely the numbers 3, 2, and 4 with the following description:

3 represents the letter E 2 represents the letter D 4 represents the letter F (because F is the letter after E, where E is represented by the number 3)

So the formula of the equation CDE=EDF is 123=324.  

Well,

Having found the formula of the equation CDE=EDF above, now we go to the answer key or solution to the problem EFH = ?

1 2 3 = 3 2 4

Using the same formula, the description is as follows:

E is represented by the number 1 F represented by the number 2 H represented by the number 3

So find the answer we are looking for, namely the value of the numbers 3, 2, and 4 above, namely:

3 represents the letter H 2 represents the letter F 4 represents the letter I (because I is the letter after H, where H is represented by the number 3)
