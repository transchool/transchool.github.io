12. what is (x+y) ^{2} = 9 sana may maka answer


Answer:

(x+y)² = 9 is a quadratic equation.

Shortcut Solution version 1:

(x+y)² = 9

take square root √ of both sides

[tex]\sqrt{(x+y)^2} = \sqrt{9}[/tex]

x + y = √9

x + y = ± 3

so,

x = -3 - y

and

x = 3 - y

Shortcut Solution version 2:

(x+y)² = 9  

apply a²-b² = (a+b)(a-b) rule,

(x+y)² - (3)² = 0

(x+y+3)(x+y-3) = 0

so

(x+y+3) = 0

x = -3 - y

and

(x+y-3) = 0

x = 3 -y

Long Cut Solution:

(x+y)² = 9

expand left side using perfect square rule,

x² + 2xy + y² = 9

x² + 2xy + y² - 9 = 0

now use Quadratic Equation Formula to solve for the roots,

and you will end up with same answer:

x = -3 - y

and

x = 3 - y
