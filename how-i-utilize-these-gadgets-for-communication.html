Gadgets for Communication

In today's world, gadgets are an important tool in communications. The following are some of the examples of gadgets and the way we could utilize them for communication purposes:

Cellphone - cellphones are the most portable gadget right now. We could install various applications such as messenger, etc that are useful in communications especially in online learning Laptops - laptops are more portable than the traditional computer set up and offers the same functionalities including communication purposes such as using skype Desktop computers - this is ideal for home set up as it offers the same functions as laptops but is cheaper and more affordable. It could be used for communication applications such as emails, messenger, etc.
