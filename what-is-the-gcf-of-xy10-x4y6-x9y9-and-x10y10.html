x*y^10=x*y*y*y*y*y*y*y*y*y*y

x^4*y^6=x*x*x*x*y*y*y*y*y*y

x^9*y^9=x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y

x^10*y^10=x*x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y*y

now we find which factor is common in all monomials

let's check all the x terms first

x*y^10=x*y*y*y*y*y*y*y*y*y*y

x^4*y^6=x*x*x*x*y*y*y*y*y*y

x^9*y^9=x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y

x^10*y^10=x*x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y*y

so an x is common,

now let's check y terms

x*y^10=x*y*y*y*y*y*y*y*y*y*y

x^4*y^6=x*x*x*x*y*y*y*y*y*y

x^9*y^9=x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y

x^10*y^10=x*x*x*x*x*x*x*x*x*x*y*y*y*y*y*y*y*y*y*y

so y^6 is common for all monomials,

now we multiply the common factors
