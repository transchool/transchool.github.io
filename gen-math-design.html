14. evaluate functions gen math​


Answer:

Evaluating a function means finding the value of f(x) =… or y =… that corresponds to a given value of x. To do this, simply replace all the x variables with whatever x has been assigned. For example, if we are asked to evaluate f(4), then x has been assigned the value of 4.

math. And when I don’t get something, now I try all different kinds of strategies. I am now able to solve a problem before I ask somebody for help. But now I don’t give up on problems until I understand them.

Although math was a blur, I knew I had to get it done, whether it was by asking for help or trying to get it done. I now know that I am somebody who always tries and won’t give up.
