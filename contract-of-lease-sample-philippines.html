13. A contract requires lease payments of $800 at the beginning of every month for 6 years. a. What is the present value of the contract if the lease rate is 4.25% compounded annually?( Round to the nearest cent )b. What is the present value of the contract if the lease rate is 4.25% compounded monthly?​


Answer:

a. To find the present value of the contract if the lease rate is 4.25% compounded annually, we can use the formula for the present value of an annuity:

PV = PMT x [1 - (1 + r)^(-n)] / r

where PV is the present value, PMT is the lease payment, r is the interest rate per period, and n is the total number of periods.

In this case, PMT = $800, r = 0.0425 (since the lease rate is given as an annual rate), and n = 6 x 12 = 72 (since there are 12 months in a year and the lease payments are made monthly for 6 years).

Plugging in the values, we get:

PV = $800 x [1 - (1 + 0.0425)^(-72)] / 0.0425

PV = $800 x 46.1799 / 0.0425

PV = $867,186.38

Therefore, the present value of the contract is approximately $867,186.38 if the lease rate is 4.25% compounded annually.

b. If the lease rate is 4.25% compounded monthly, we need to adjust the interest rate and the number of periods accordingly. The interest rate per month is r/12 = 0.0425/12 = 0.00354, and the total number of periods is n = 6 x 12 = 72 (as before).

Using the same formula as before, we get:

PV = $800 x [1 - (1 + 0.00354)^(-72)] / 0.00354

PV = $800 x 46.9196 / 0.00354

PV = $10,528,196.61

Therefore, the present value of the contract is approximately $10,528,196.61 if the lease rate is 4.25% compounded monthly.

Answer:

a. To find the present value of the contract if the lease rate is 6.12% compounded annually, we can use the formula for the present value of an annuity:

PV = PMT x ((1 - (1 + r)^-n) / r)

Where PV is the present value, PMT is the payment amount, r is the interest rate per period, and n is the number of periods.

In this case, PMT = $500, r = 6.12% / 12 = 0.51% per month, and n = 8 x 12 = 96 months.

Plugging these values into the formula, we get:

PV = $500 x ((1 - (1 + 0.0051)^-96) / 0.0051) = $44,422.68

Therefore, the present value of the contract is approximately $44,422.68.

b. To find the present value of the contract if the lease rate is 6.12% compounded daily, we can use the formula:

PV = PMT x ((1 - (1 + r/365)^-(n x 365)) / (r/365))

Where PV is the present value, PMT is the payment amount, r is the annual interest rate, and n is the number of years.

In this case, PMT = $500, r = 6.12%, and n = 8.

To convert the annual interest rate to a daily rate, we can use the formula:

r per day = (1 + r per year)^(1/365) - 1

r per day = (1 + 0.0612)^(1/365) - 1 = 0.00015888

Plugging these values into the formula, we get:

PV = $500 x ((1 - (1 + 0.00015888)^-(8 x 365)) / (0.00015888)) = $44,462.78

Therefore, the present value of the contract if the lease rate is 6.12% compounded daily is approximately $44,462.78.

30. A lease is classified as a finance lease if?


Answer:

A lease is normally classified as a finance lease if any of the following conditions apply: The asset transfers to the lessee at the end of the lease term. The lessee has an option to purchase the asset from the lessor at below fair value. The lease term is for a significant part of the asset's useful economic life.
