6. Mag bigay ng meaning 1. Bookmarking 2. Book mark 3. Cloud4. Web page 5. Web siteTagalog po plas​


Answer:

1. Bookmarking - ito ay talaan ng address ng isang web page sa iyong computer upang madali mo itong mahanap muli.

2. Bookmark - ang bookmark ay nagsisilbi lamang bilang isang shortcut pabalik sa website.

3. Cloud - ang cloud ay ang paghahatid ng iba't ibang serbisyo sa pamamagitan ng Internet, kabilang ang pag-iimbak ng data, mga server, database, networking, at software.

4. Web page - ang Web page ay isang dokumento para sa World Wide Web na isang natatanging unipormeng resource locator (URL).

5. Web site - isang hanay ng mga kaugnay na web page na matatagpuan sa ilalim ng iisang domain name, na karaniwang ginagawa ng isang tao o organisasyon.

13. Paki explain naman po ng food chain and food web sa tagalog po. Salamat po.


       Ang food chain ay nagpapakita ng payak ngunit tiyak na ugnayan sa pagitan ng  mga producer at mga consumer.

       Ang food web ay naglalarawan ng ilang dugtung-dugtong na food chain.

         Makikita sa mga food web kung ano ang nangyayari sa kalikasan, kung saan  may malawak na ugnayan ang mga organismo sa pagkuha ng kani-kanilang  pagkain sa kanilang kapaligiran.

        Ang food web ay pangkat ng mga magkakarugtong na food chain. Ang food web  ay nabubuo mula sa magkakaibang food chain kung ang primary consumer, tulad ng  uod, ay maaaring kainin hindi lang ng ibon, kundi ng iba pang mga hayop na bahagi ng  iba pang food chain. Samakatuwid, ang food web ay nagpapakita ng mas malawak na  larawan kung paano naisasalin ang enerhiya sa ating kalikasan.
