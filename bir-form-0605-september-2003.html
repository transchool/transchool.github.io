4. Set Builder form C=April,June,september and november}


Answer:

C = { April, June, September and November }

C = { x : x is a month with 30 days }

Step-by-step explanation:

Set Builder Form or Rule Method

Set builder form or rule method represents a set, with a property, such that all elements in the set satisfy that property.

The list of the elements in a set is called as roster.

How do we express set builder form or rule method?

In this method we write a general element x and ‘:’ meaning such that and state its property within brackets.

Examples:

Roster: x = { a, e, i, o, u }

Set builder form: { x: x is a vowel }

A = { 2, 4, 6, 8 }

A = { x : x is an even natural number, x ≤ 8 }
