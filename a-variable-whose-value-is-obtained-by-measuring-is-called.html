Discrete Variables are variables in the form of categorizing data or distinguishing or grouping certain types. Usually the data for this type of variable is obtained from the calculation results. Data for discrete variables are often called nominal data or dichotomous data. For example: Dichotomous data 1 for the correct category and 0 for the wrong category.

Discrete variables can be calculated in limited time. For example, you can count the change in your pocket. You can count the money in your bank account. You can also calculate the amount of money in each person's bank account. It may take you a long time to calculate that last item, but the point is—it still counts.

A continuous variable is a variable whose data can be operated mathematically. Data for continuous variables is obtained from the measurement process so that the data can be in the form of fractions or in decimal form, for example, 3.5 meters, value 7.5, average 7.0 and others. In general, continuous variables have values ​​which are the results of measurements. Examples of data for these variables are test scores, rankings, height, weight, length, distance. If grouped, data for continuous variables can be classified as ordinal data, interval data and ratio data.
