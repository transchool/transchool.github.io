Find the distance between the two given points.

1. S (-1, 6) and T (-1, 14)

[tex]\[\begin{array}{l}d = \sqrt {{{({x_2} - {x_1})}^2} + {{({y_2} - {y_1})}^2}} \\\\d = \sqrt {( - 1 - {{( - 1)}^2} + {{(14 - 6)}^2}} \\\\d = \sqrt {{{(0)}^2} + {{(8)}^2}} \\\\d = \sqrt {0 + 64} \\\\d = \sqrt {64} \\\\d = 8\end{array}\][/tex]

2. H (-9, 3) and G (6, 3)

[tex]\[\begin{array}{l}d = \sqrt {{{({x_2} - {x_1})}^2} + {{({y_2} - {y_1})}^2}} \\\\d = \sqrt {(6 - {{( - 9)}^2} + {{(3 - 3)}^2}} \\\\d = \sqrt {{{(15)}^2} + {{(0)}^2}} \\\\d = \sqrt {225 + 0} \\\\d = \sqrt {225} \\\\d = 15\end{array}\][/tex]

3. M (0, 0) and N (4, 6)

[tex]\[\begin{array}{l}d = \sqrt {{{({x_2} - {x_1})}^2} + {{({y_2} - {y_1})}^2}} \\\\d = \sqrt {{{(4 - 0)}^2} + {{(6 - 0)}^2}} \\\\d = \sqrt {{{(4)}^2} + {{(6)}^2}} \\\\d = \sqrt {16 + 36} \\\\d = \sqrt {52} \\\\d = 7.21\end{array}\][/tex]

4. U (-5, 2) and P (-5, 7)

[tex]\[\begin{array}{l}d = \sqrt {{{({x_2} - {x_1})}^2} + {{({y_2} - {y_1})}^2}} \\\\d = \sqrt {{{( - 5 - ( - 5))}^2} + {{(7 - 2)}^2}} \\\\d = \sqrt {{{(0)}^2} + {{(5)}^2}} \\\\d = \sqrt {0 + 25} \\\\d = \sqrt {25} \\\\d = 5\end{array}\][/tex]

5. A (-3, 2) and M (9, 7)​

[tex]\[\begin{array}{l}d = \sqrt {{{({x_2} - {x_1})}^2} + {{({y_2} - {y_1})}^2}} \\\\d = \sqrt {{{(9 - ( - 3))}^2} + {{(7 - 2)}^2}} \\\\d = \sqrt {{{(12)}^2} + {{(5)}^2}} \\\\d = \sqrt {144 + 25} \\\\d = \sqrt {169} \\\\d = 13\end{array}\][/tex]
