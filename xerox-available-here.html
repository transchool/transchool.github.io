10. 2. A Xerox machine can be rented for P14,000 for seven days , but for the eighth day onwards , an additional cost of P1,000 per day is added . Represent the cost of renting a Xerox machine as a piecewise function of the number of days ( d ) it is rented . pasaguy po. need k ngayun plssss ​


Step-by-step explanation:

The problem states that renting a xerox machine would cost a flat 14000 for seven days. That is why in the first equation, this part of the problem is represented as a flat 14000, if 0 is less than (since the number of days cant be negative) d is less than or equal to 7 (flat 14000 if within 7 days).

The second part of the problem states that for the eight day onwards, 1000 per day is added. This is why in the second equation, the problem is represented as 14000 + (1000(d-7)), if d is equal to or greater than 8. 14000 represents the flat rate, (d-7) represents the extra days only since the 7 days is covered in the flat rate, 1000(d-7) represents 1000 multipled by the extra days, if x is greater than or equal to 8 indicates that the equation should be used when the number of days that it would be rented is 8 or more days.
