12. The GPTA officers can paint a classroom in 3 hours, while Class officers can paint the same classroom in 5 hours. If the GPTA and Class officers work together, how long will i take them to paint the classroom?​


Answer:

1.875 hours

Step-by-step explanation:

Use the Working Together Formula:

1/T = 1/A + 1/B

where A = time of GPTA finishing the work

B = time of Class Officers finishing the work

T = total time of them working together

1/T = 1/3 + 1/5

1/T = (5+3)/15

1/T = 8/15

Cross multiply:

(8)(T) = (1)(15)

8T = 15

T = 15/8 or 1.875 hours
