10. using a techans triangle what is the formula in finding a percentage​


The Techan's Triangle is a set of three formulas, similarly arranged as SOHCAHTOA. The formulas are:

1. Base = Rate x Percentage

2. Rate = Percentage / Base x 100 (and don't forget the percent sign %)

3. Percentage = Base x Rate
