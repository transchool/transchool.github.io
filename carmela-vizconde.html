Answer:

₱69

Step-by-step explanation:

Let x = starting money

Carmela lent 35 to Lester

x - 35

Carmela received 50

x - 35 + 50

Sister borrowed 20

x - 35 + 50 - 20

Paid 18

x - 35 + 50 - 20 - 18

Still had 43

x - 35 + 50 - 20 - 18 = 43

43 = x - 35 + 50 - 20 - 18

43 = x- 23

43 + 26 = x

69 = x

2. Marie earned 75 an hour and Carmela earned 83 an hour. Marie and Carmela worked the same number of hours. Marie earned 525,how much did Carmela earned? ​


Answer:

If Marie earned 525, Let n be Carmela's

75 : 525 = 83 : n

[tex] \frac{75}{525} = \frac{83}{n} [/tex]

Apply cross multiplication.

[tex]75n = 43575[/tex]

Let's divide both sides by 75.

[tex] \frac{75n}{75} = \frac{43575}{75} \\ \\ \\ n = 581[/tex]

Therefore, Carmela earned 581
